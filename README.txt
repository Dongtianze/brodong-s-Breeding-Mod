阶段目标

性别与繁殖
1、实现性别属性，实现异性匹配繁殖（制作新的BreedGoal及AgeableAnimal）
2、对于胎生动物，增加怀孕状态；对于卵生动物，增加孵蛋，删除交配即孵化小鸡（将鸡蛋机制改为类似海龟蛋机制）

基因与性状
1、动物增加基因序列，通过基因决定动物的血条、产出等数值（以及视觉效果）
2、动物繁殖的后代会组合亲代的基因。
3、随机出现的基因突变

饲养与生命
1、为动物增加饱食度，饱食度为空时会扣血直到死亡，动物会自动寻找食物（草、胡萝卜等）
2、增加饲槽，动物会优先在饲槽进食
3、删除原版饲喂即inLove的逻辑，改为饱食度高且不在冷却时概率进入寻偶状态
4、对于可以饲养的动物，加入最高寿命，寿命到时即死亡

群居与生活
1、增加动物结成群体的机制，动物群体会一起行动，离队太远会脱群
2、增加鞭子等工具，玩家可以用鞭子放牧，将动物群驱使到目标地方
3、为狗加入牧羊的功能，可以将离群的羊赶回群中

更多细节
动物会产生粪便，粪便分解过程中会对周围一定范围内的随机一个方块施加骨粉效果，玩家也可以铲起粪便堆肥成骨粉
